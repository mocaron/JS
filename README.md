# JS Mini Game - Online Shopping

## 사용언어
Html, Css, Vanila JS

## 과제
* 자바스크립트를 사용해 동적으로 데이터 받아오기
* 버튼을 누르면 목록 filtering
* 각 버튼에 hover 기능
